## **3. Problem Statement**

Despite having:

- Microsoft 365 Business Standard  
- Microsoft 365 Business Basic  
- Admin permissions  
- A U.S. tenant  
- Self‑service purchases enabled  

…the user was still unable to purchase Power BI Premium Per User due to:

### **A billing system mismatch (MCA vs Microsoft 365 Commerce)**

This mismatch creates a **dead‑end** where:

- “Purchase services” is missing  
- PPU cannot be purchased  
- Pro cannot be purchased  
- Copilot cannot be purchased  
- Support articles don’t match the UI

## **5. SQL Analysis**

### **Total time lost**
```sql
SELECT SUM(time_lost_min) AS total_minutes_lost
FROM licensing_attempts
WHERE result = 'Failed';
```

### **Most common error**
```sql
SELECT error_message, COUNT(*) AS occurrences
FROM licensing_attempts
GROUP BY error_message
ORDER BY occurrences DESC;
```

## **6. Python Analysis**

```python
import pandas as pd
import matplotlib.pyplot as plt

attempts = pd.read_csv("licensing_attempts.csv")
frustration = pd.read_csv("frustration_scores.csv")

total_time = attempts[attempts['result'] == 'Failed']['time_lost_min'].sum()
print("Total time lost:", total_time)

plt.figure(figsize=(8,5))
plt.bar(attempts['license'], attempts['time_lost_min'], color='orange')
plt.xticks(rotation=45)
plt.title("Time Lost per Licensing Attempt")
plt.ylabel("Minutes")
plt.tight_layout()
plt.show()

plt.figure(figsize=(8,5))
plt.plot(frustration['step'], frustration['frustration_score'], marker='o')
plt.xticks(rotation=45)
plt.title("Frustration Score Over Licensing Journey")
plt.ylabel("Score (1-10)")
plt.tight_layout()
plt.show()
```

## **9. Insights & Findings**

- Licensing failures stem from **system fragmentation**, not user error.  
- MCA billing accounts cannot purchase enterprise add‑ons.  
- Admin center UI does not communicate billing limitations.  
- Users lose significant time troubleshooting issues that require support escalation.

- Powerful ecosystem  
- Deep integration  
- Enterprise‑grade security  
- Flexible licensing  

### **Cons**
- Fragmented billing systems  
- Vague error messages  
- UI inconsistencies  
- High friction for small businesses  

---

## **11. Recommendations**

### **For Microsoft**
- Unify billing systems  
- Improve error messaging  
- Add guided troubleshooting  
- Add “Convert billing account” button  

### **For Users**
- Verify billing account type early  
- Document licensing attempts  
- Contact support sooner  
- Consider alternative tools
