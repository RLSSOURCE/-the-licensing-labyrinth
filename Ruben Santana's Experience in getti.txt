# ⭐ **2. SQL Analysis**

### **Total time lost**
```sql
SELECT SUM(time_lost_min) AS total_minutes_lost
FROM licensing_attempts
WHERE result = 'Failed';
```

### **Most common error**
```sql
SELECT error_message, COUNT(*) AS occurrences
FROM licensing_attempts
GROUP BY error_message
ORDER BY occurrences DESC;
```

### **Average frustration score**
```sql
SELECT AVG(frustration_score) AS avg_frustration
FROM frustration_scores;
```

# ⭐ **3. Python Analysis (Pandas + Matplotlib)**

```python
import pandas as pd
import matplotlib.pyplot as plt

# Load data
attempts = pd.read_csv("licensing_attempts.csv")
frustration = pd.read_csv("frustration_scores.csv")

# Total time lost
total_time = attempts[attempts['result'] == 'Failed']['time_lost_min'].sum()
print("Total time lost:", total_time)

# Bar chart: time lost per attempt
plt.figure(figsize=(8,5))
plt.bar(attempts['license'], attempts['time_lost_min'], color='orange')
plt.xticks(rotation=45)
plt.title("Time Lost per Licensing Attempt")
plt.ylabel("Minutes")
plt.tight_layout()
plt.show()

# Line chart: frustration over steps
plt.figure(figsize=(8,5))
plt.plot(frustration['step'], frustration['frustration_score'], marker='o')
plt.xticks(rotation=45)
plt.title("Frustration Score Over Licensing Journey")
plt.ylabel("Score (1-10)")
plt.tight_layout()
plt.show()
```

# ⭐ **5. Full Case Study (Professional Tone)**

## **Executive Summary**
This case study explores the complexity of Microsoft’s licensing system through a simulated attempt to activate Power BI Premium Per User (PPU). Despite having valid Microsoft 365 Business licenses, the user encountered multiple blockers caused by a billing system mismatch (MCA vs Microsoft 365 Commerce).

The study uses mock data, SQL/Python analysis, and UX mapping to identify root causes and propose improvements.

## **Key Findings**
- Licensing failures stem from **system fragmentation**, not user error.  
- MCA billing accounts cannot purchase enterprise add‑ons.  
- Admin center UI does not clearly communicate billing limitations.  
- Users lose significant time troubleshooting issues that require support escalation.  

## **Recommendations**
- Unify billing systems  
- Improve error messaging  
- Provide guided troubleshooting  
- Add a “Convert billing account” button
